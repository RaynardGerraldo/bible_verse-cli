# bible_verse-cli

<h1 align="center">Bible verses on the command line</h1>

### Dependencies
``` pip install opencv-python ```

### Usage

```
Usage:
  bible_verse [BOOK CHAPTER VERSE]
  bible_verse [-i [BOOK CHAPTER VERSE]]
  bible_verse [-r -i] | [-i -r]
  Options:
	-r : Get random verse
	-i : Write verse to image
```
